# Notebook Index

이 문서는 `전략_notebook` 폴더의 노트북을 취업용 포트폴리오 관점에서 빠르게 파악할 수 있도록 정리한 목록입니다.

## 01. 고배당 + 저 DPS 변화율 전략

- 경로: `전략_notebook/01_고배당_저DPS변화율_전략/고배당전략_DPS변화율필터.ipynb`
- 목적: 배당 관련 지표, DPS 변화율, IVOL 필터를 활용해 고배당 포트폴리오를 구성합니다.
- 주요 처리: 수정 DPS, 수정종가, 시장구분, 배당반영수정주가, IVOL, 팩터 데이터를 활용합니다.
- 공개 전 점검: 기존 로컬 절대경로를 상대경로로 바꿔야 합니다.

## 02. PBR + ROE 전략

- 경로: `전략_notebook/02_PBR_ROE_전략/pbr+roe_new.ipynb`
- 목적: 저 PBR과 고 ROE를 결합해 밸류/퀄리티 성격의 종목군을 구성합니다.
- 주요 처리: 소형주 필터링, PBR 순위, ROE 순위, KOSPI200 벤치마크 비교, 팩터 회귀를 포함합니다.
- 공개 전 점검: 일부 입력 파일명이 로컬 작업 파일명 기준입니다.

## 03. VIP Score 전략

- 경로: `전략_notebook/03_VIP_score_전략/VIP_Score.ipynb`
- 목적: Value, Investment, Profitability 요소를 결합해 종합 점수 기반 포트폴리오를 구성합니다.
- 주요 처리: 수정주가, 시가총액, PBR, 영업이익, 자산총계 데이터를 활용해 점수 패널을 생성합니다.
- 공개 전 점검: 생성 CSV는 커밋하지 않고 재생성 산출물로 관리합니다.

## 04. 검색량 데이터 전략

- 경로: `전략_notebook/04_검색량데이터_전략/Empirical Analysis.ipynb`
- 목적: 검색량 지표의 계절성을 제거한 ASVI를 계산하고 배당 포트폴리오 및 회귀분석에 적용합니다.
- 주요 처리: SVI 로그 차분, 월 더미 기반 계절성 제거, 배당 포트폴리오 구성, 투자주체별 순매수강도, 최종 실증분석을 포함합니다.
- 관련 노트북: `삼성전자_검색량데이터_전략.ipynb`

## 05. 고베타 + 레짐 반영 전략

- 경로: `전략_notebook/05_고베타_레짐반영_전략/Beta&regime_코스피_코스닥.ipynb`
- 목적: 베타와 시장 레짐을 반영해 KOSPI/KOSDAQ 종목군의 동적 포트폴리오를 테스트합니다.
- 주요 처리: KOSPI, KOSDAQ, beta, 일별수정주가 데이터를 활용합니다.
- 공개 전 점검: 노트북 출력 크기가 큰 편이므로 필요하면 결과 이미지만 남기고 출력 정리를 고려합니다.

## 06. Value-Quality 레짐 동적 배분

- 경로: `전략_notebook/06_Value_Quality_전략/Value_quality_레짐동적배분.ipynb`
- 목적: 시장 레짐에 따라 Value/Quality 포트폴리오를 동적으로 배분합니다.
- 주요 처리: 월수익률 계산, 종목 추출, NAV 계산, KOSPI200 벤치마크 비교, 성과 지표 계산을 포함합니다.
- 공개 전 점검: 기존 로컬 절대경로를 상대경로로 바꿔야 합니다.

## 공통 주의사항

- 원천 데이터는 공개 저장소에 포함하지 않습니다.
- 재현용 샘플 데이터가 필요하면 `data/processed/sample_*.csv` 형태로 별도 생성 후 민감성/라이선스를 확인합니다.
- 노트북 일부는 기존 연구 흐름을 보존한 상태라서, 완전한 재현성을 위해서는 데이터 경로와 파일명을 통일하는 후속 정리가 필요합니다.
